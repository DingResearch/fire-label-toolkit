SatFire Labeler is a toolkit for labeling, reviewing, and validating wildfire detections in satellite imagery datasets. The repository provides tools to assist researchers in generating high-quality annotations and verifying existing labels used for wildfire detection, prediction, remote sensing analysis, and machine learning model development.

Accurate labeling of wildfire events in satellite data is critical for building reliable detection and prediction systems and benchmarking wildfire monitoring algorithms. This toolkit supports dataset curation workflows including annotation, quality control, and label validation across large collections of satellite imagery.

The tools in this repository are designed to support remote sensing, wildfire detection and prediction, and AI/machine learning research, enabling reproducible dataset preparation and improved label quality.
